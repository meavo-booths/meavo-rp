# Examples

Do **not** copy example repos into new projects. Use them as **reference** when filling placeholders.

| Repo | Good reference for |
|------|-------------------|
| [meavo-rp](https://github.com/meavo-booths/meavo-rp) | Next.js App Router, domain layer, sheet sync, personas, `docs/domain.md` depth |
| [meavo-db](https://github.com/meavo-booths/meavo-db) | Schema-owned repo, `data-model.md` as primary doc |
| [meavo-gateway](https://github.com/meavo-booths/meavo-gateway) | Auth, tool cards, cron patterns |
| [sales](https://github.com/meavo-booths/sales) | Next.js + Sheets + shared navigation scaffold |

When bootstrapping, **discover the target repo first** — then borrow *patterns*, not prose.
