# Legacy Google Apps Script reference

Copy the current Apps Script project files here for porting:

- `WebAppLogic.js`
- `LoggerLogic.js`, `Logger.html`
- `IpLogger.html`
- `index.html`
- `CatalogueLogic.js`
- `Login.js`
- `Other scripts/*.js`
- `appsscript.json`

Do **not** deploy GAS from this repo after production cutover.

Use `clasp pull` from the live Apps Script project, or copy from your local `Meavo-RP-App` folder.
